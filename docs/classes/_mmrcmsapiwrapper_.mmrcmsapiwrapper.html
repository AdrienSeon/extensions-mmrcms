getViewMoreItems

  • getViewMoreItems(source: Source, homepageSectionId: string, metadata: any, resultPageLimiter?: undefined | number): Promise<MangaTile[] | null>

  • Inherited from MmrcmsAPIWrapper.getViewMoreItems

    • Defined in node_modules/paperback-extensions-common/dist/APIWrapper.d.ts:22

    Performs a 'get more' request. Usually this is done when a homesection has it's 'View More' button tapped, and the user is starting to scroll through all of the available titles in each section. It is recommended that when you write your tests for a source, that you run one test using this function, for each homepageSectionId that the source offers, if those sections are expected to traverse multiple pages

    Parameters

    • source: Source
    • homepageSectionId: string
    • metadata: any
    • Optional resultPageLimiter: undefined | number

      How many pages this should attempt to iterate through at most. This prevents you from being in an infinite loop. Defaults to 3.

    Returns Promise<MangaTile[] | null>
